## Tools

### Web Application Testing

**Proxy Tools**
- [Burp Suite](https://portswigger.net/burp) - Industry standard
- [OWASP ZAP](https://www.zaproxy.org/) - Free alternative
- [Caido](https://caido.io/) - Modern alternative

**Scanners**
- [Nuclei](https://github.com/projectdiscovery/nuclei) - Fast vulnerability scanner
- [SQLMap](https://sqlmap.org/) - SQL injection tool
- [Nikto](https://github.com/sullo/nikto) - Web server scanner
- [WPScan](https://wpscan.com/) - WordPress scanner

**Reconnaissance**
- [Amass](https://github.com/OWASP/Amass) - Subdomain enumeration
- [Subfinder](https://github.com/projectdiscovery/subfinder) - Subdomain discovery
- [httpx](https://github.com/projectdiscovery/httpx) - HTTP toolkit
- [Nmap](https://nmap.org/) - Network scanner
- [masscan](https://github.com/robertdavidgraham/masscan) - Fast port scanner

**API Testing**
- [Postman](https://www.postman.com/) - API development
- [Insomnia](https://insomnia.rest/) - API client
- [Arjun](https://github.com/s0md3v/Arjun) - HTTP parameter discovery

**Fuzzing**
- [ffuf](https://github.com/ffuf/ffuf) - Fast web fuzzer
- [wfuzz](https://github.com/xmendez/wfuzz) - Web fuzzer
- [Gobuster](https://github.com/OJ/gobuster) - Directory brute forcing

**Exploitation**
- [Metasploit](https://www.metasploit.com/) - Exploitation framework
- [Exploit Database](https://www.exploit-db.com/) - Exploit archive

**Mobile Security**
- [MobSF](https://github.com/MobSF/Mobile-Security-Framework-MobSF) - Mobile security testing
- [Frida](https://frida.re/) - Dynamic instrumentation
- [Objection](https://github.com/sensepost/objection) - Mobile exploration

**Cloud Security**
- [ScoutSuite](https://github.com/nccgroup/ScoutSuite) - Multi-cloud security
- [Prowler](https://github.com/prowler-cloud/prowler) - AWS security
- [CloudSploit](https://github.com/aquasecurity/cloudsploit) - Cloud security
